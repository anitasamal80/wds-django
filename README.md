# Django Project - AWDS

- An authenticated web data service using Django, SQLite and Plotly.It shows employee rate chart year on year.

## Features

- Visualize data using Plotly
- User authentication

## Installation

For Windows:

```bash
run_wds.bat
```

For Linux:

```bash
./run_wds.sh
```

The default development server IP should be http://localhost:8000/
